The Difference Between CSMA/CD and CSMA/CA

Edited on 2018-12-28 | In Computer Networking

Network Links

As we know, there are two types of network links: point-to-point links and broadcast links.

A point-to-point link consists of a single sender at one end of the link and a single receiver at the other end of the link.

A broadcast link can have multiple sending and receiving nodes all connected to the same, single, shared broadcast channel. The term broadcast is used here because when any one node transmits a frame, the channel broadcasts the frame and each of the other nodes receives a copy.

There is a multiple access problem in broadcast links. Because all nodes are capable of transmitting frames, more than two nodes can transmit frames at the same time. When this happens, all of the nodes receive multiple frames at the same time; that is, the transmitted frames collide at all of the receivers and the frames become corrupted.

Solutions to Multiple Access Problems

There are three kinds of methods to solve the multiple access problems:

  • Channel Partition
  • Random Access
  • Taking Turn
